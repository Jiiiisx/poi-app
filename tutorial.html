Selamat Datang di Tutorial Dashboard Point of Interest!

Dashboard ini dirancang untuk membantu Anda mengelola dan memantau data calon pelanggan (Point of Interest) secara efisien.

1. Login

Untuk mengakses dashboard, Anda harus login menggunakan akun Google Anda. Cukup klik tombol "Sign in with Google" dan pilih akun Anda.

2. Tampilan Utama (Team Telda)

Setelah login, Anda akan melihat tampilan utama "Team Telda".

  • Ringkasan Statistik: Di bagian atas, Anda akan melihat ringkasan jumlah total Point of Interest (POI), serta jumlah POI yang dikategorikan sebagai Sekolah, Non-Sekolah, dan Pemerintahan.
  • Daftar Calon Pelanggan: Ini adalah tabel utama yang menampilkan semua data calon pelanggan. Anda dapat mencari data menggunakan kolom pencarian di atas tabel.
  • Filter Data: Anda dapat memfilter data berdasarkan kategori:
    • Semua: Menampilkan semua data.
    • Sekolah: Menampilkan data POI sekolah.
    • Non-Sekolah: Menampilkan data POI selain sekolah.
    • Pemerintahan: Menampilkan data POI pemerintahan.
  • Menambah Data: Klik tombol Add Calon Pelanggan Baru untuk membuka form dan menambahkan data pelanggan baru.
  • Mengubah & Menghapus Data: Di setiap baris data, terdapat tombol "Actions" untuk mengubah (edit) atau menghapus data.

3. Tampilan Monitoring (Team Non-Telda)

Anda dapat beralih ke tampilan "Team Non-Telda" untuk memantau kinerja sales dan pelanggan yang sudah ada.

  • Pilih Sales: Di sidebar kiri, pilih nama sales untuk melihat data pelanggan yang mereka tangani.
  • Monitoring Pelanggan: Anda akan melihat daftar pelanggan beserta informasi penting seperti:
    • Status Billing: Status pembayaran tagihan bulanan (misalnya, PAID, UNPAID).
    • Redaman Loss: Tingkat redaman sinyal. Nilai di bawah -23 dB akan ditandai merah.
    • FUP: Penggunaan kuota internet (Fair Usage Policy).
  • Ringkasan Kinerja: Di bagian atas, Anda akan melihat ringkasan kinerja sales yang dipilih, termasuk total pelanggan dan rata-rata redaman.

4. Fitur Umum

  • Pencarian: Gunakan kolom pencarian untuk menemukan data dengan cepat di setiap tampilan.
  • Logout: Klik tombol "Logout" di sidebar untuk keluar dari dashboard.
